## 2025-05-14 - Initial UX/Accessibility Audit
**Learning:** The application had several accessibility gaps including unlinked form labels, missing `aria-label` on icon-only interactive elements, and lack of visual feedback for asynchronous operations (contact form).
**Action:** Implement `aria-label`, link labels with `id`/`htmlFor`, and add loading/success states to improve the user experience and accessibility.
